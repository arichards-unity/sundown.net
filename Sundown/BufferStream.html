BufferStream Class

A wrapper of the Buffer exposing the functionality as a Stream.

SundownNet : Sundown Namespace

Syntax

public class BufferStream : System.IO.Stream

Remarks

This class enables the developer to use buffers as Streams utilizing various other tools to fill the Buffers with data.
